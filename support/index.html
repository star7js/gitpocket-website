Frequently Asked Questions

How do I sign in?

Tap the "Sign in with GitHub" button on the welcome screen. You'll be redirected to GitHub to authorize GitPocket. Once authorized, you'll be signed in automatically.

Is my GitHub token secure?

Yes. Your OAuth token is stored in iOS Keychain, which uses hardware-level encryption. The token never leaves your device and is only used to communicate directly with GitHub's API.

Can I use GitPocket offline?

Yes, to an extent. GitPocket caches repository data, gists, and releases locally so you can browse previously viewed content offline. However, you need an internet connection to fetch new data or perform actions like creating gists.

How do I create a gist from another app?

Use the iOS Share Extension: Select text in any app → tap Share → choose GitPocket. You can then create a new gist with the selected content.

Does GitPocket support GitHub Enterprise?

Not currently. GitPocket only supports github.com accounts at this time. GitHub Enterprise support may be added in a future update.

How do I sign out?

Go to Settings (tap your profile icon) → scroll down → tap "Sign Out". This will remove your OAuth token and clear all cached data.

Why can't I push code or create repositories?

GitPocket is currently a read-only client with limited write capabilities (gists only). Full repository management features may be added in future updates based on user feedback.
